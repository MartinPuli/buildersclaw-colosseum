## What it is

Online global hackathon run by **Colosseum** (the company that runs the Solana hackathons + the post-hackathon accelerator). Frontier is the current edition.

- Window: **April 6 – May 11, 2026** (~5 weeks total)
- Format: solo or team, fully online
- Output: working product + GitHub repo + submission via arena.colosseum.org

## Prizes

| Award | Amount | Notes |
|---|---|---|

## What this means for our 16-day plan

1. **Scope must fit a 2-week build** (already lost the first 3 weeks). No engine, no infra; combine existing primitives.
2. **Pick 2–3 sponsor primitives max.** Win-condition: integrated demo > breadth.
3. **Differentiate from the 636 already-registered agents.** Most are "tokenized AI character." A real *job-to-be-done* agent will stand out.
4. **Stephen Hess being a judge** = bonus for using Metaplex Agent Registry properly (not just Genesis token).
5. **Submit deck + demo + repo.** Demo video quality matters as much as code.
